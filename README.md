# Taller_1_aplicaciones_con_IA

Primer paso, crear el entorno virtual mediante el comando: python -m venv env
Segundo, activar el entorno virtual con el comando: .\env\Scripts\Activate
Tercero, instalamos las librerias que se encuentran requirments.txt usando el comando: pip install .\requirements.txt
Cuarto, hay que agregar en algunos casos el archivo .env con la API_KEY
Por ultimo, iniciar el programa mediante: python .\app_gemini.py
